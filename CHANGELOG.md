# Change Log

## 1.0.0
- Initial stable release
- MPEG-DASH (.mpd) and HLS (.m3u8) video playback support
- Public share link support with full playback functionality
- Shaka Player UI integration with modern video controls
- Automatic MIME type registration for .mpd and .m3u8 files
- Nextcloud 24-32 compatibility
- Responsive video player with fullscreen support
- Poster image support (automatic detection of .jpg with same name)
- Subtitle support via WebVTT (.vtt files)
- Clean, modern player interface
